Introduction

A simple REST API in Laravel

Base URL: http://localhost

Cette documentation vise à fournir toutes les informations dont vous avez besoin pour travailler avec notre API.

Au fur et à mesure que vous faites défiler, vous verrez des exemples de code pour travailler avec l'API dans différents langages de programmation dans la zone sombre à droite (ou dans le cadre du contenu sur mobile). Vous pouvez changer la langue utilisée avec les onglets en haut à droite (ou depuis le menu de navigation en haut à gauche sur mobile).

Authenticating requests
